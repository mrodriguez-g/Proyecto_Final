# Análisis Exploratorio de Datos con Python y SQL

## Objetivo

Realizar un análisis exploratorio utilizando Pandas, Plotly y DuckDB sobre un conjunto de datos de ventas.

## Dataset

Superstore Sales Dataset

Fuente:
[https://www.kaggle.com/](https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting?resource=download)

## Tecnologías

- Python
- Pandas
- NumPy
- Plotly
- DuckDB
- Google Colab

## Contenido

- Carga de datos
- Limpieza
- EDA
- Visualizaciones
- Consultas SQL

## Cómo ejecutar
1. Descargar el repositorio.
2. Abrir el notebook en Google Colab.
3. Ejecutar todas las celdas.
